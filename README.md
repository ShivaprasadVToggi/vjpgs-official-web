# VJ-PG's - Student Housing Platform

The official booking and brokerage platform for VJ-PG's, serving students of Cambridge Institute of Technology and Garden City University.

## Tech Stack
- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Deployment:** Vercel/Netlify
- **Icons:** Lucide React

## Features
- **Dynamic Listings:** Real-time PG availability.
- **WhatsApp Integration:** Direct-to-founder lead generation.
- **Image Carousel:** High-res gallery for student accommodations.
- **Blind Location Logic:** Protects brokerage revenue by masking exact addresses.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the local server.
